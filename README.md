# Feliz Quotation

An online calculator to estimate jewelry pricing for **Feliz** (14k and 18k gold).

## Features

- **Gold** — Price per gram, weight (grams), gold type with process loss (Yellow 10%, White/Rose 20%)
- **Services** — Labor (Goldsmith), Polish & Gold Plating, Laser Engrave (Rate × Qty)
- **Stones** — Add multiple stone lines: Stone Setting + Stone Price × Qty (50% margin); separate from main profit margin
- **Quotation** — Two outputs: **14k estimate** (gold −₱1,000/gram, 15% profit) and **18k** (25% profit). Raw cost, profit, stone cost, final quote, downpayment (70% if &gt;₱50k, else 50%), balance due
- Password-protected access

## Tech

React 18, TypeScript, Vite, Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

**Build for production**

```bash
npm run build
npm run preview
```
